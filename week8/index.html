CSS Transitions

Steps to Create CSS Transisions

1. Declare the original state of the element in the default style declaration.

2. Declare the final state of your transitioned element (ie a hover state)

3. Include the transition functions using the transition properties

  • transition-property: defines elements to be transformed
  • transition-duration: how long the transition will take (sec or ms)
  • transition-timing-function: controls the pace of the transition even more(ease, linear, ease-in, ease-out, ease-in-out)
  • transition-delay: allows a delay before the animation begins (in ms or sec)

CSS Animations

CSS animations allow you more control than transitions by using keyframes. A keyframe defines the start and end af a smooth transition. Adding multiple keyframs allows for more complex animations

Steps for animations

  • create a named animation
  • attach the animation to an element
  • use percentages to indicate where in the animation the keyframes are

Drag and Drop

Process

  • Set the draggable attribute on any HTML elements you’d like to be draggable (draggable="true")
  • Add an event listener for the dragstart event on any draggable HTML elements.
  • Add an event listener for the dragover and drop events on any elements that you want to have accept dropped items

The Data Transafer Object allows you to get and set data about the elements being dragged (data type and value)
