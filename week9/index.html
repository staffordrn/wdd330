Chapter 9: the Window Object

Every Javascript environment has a global object. In a browser it is called the window object. Any global variables are properties of this object and any functions you create are its methods.

Browser Object Model (BOM)

  • A collection of  properties and methods that have info about the browser and screen
  • These properties and methods are available through the window object

The Browser History

  • Window.history.length shows how many pages have been visited before the current page
  • Window.history.go() can be used to go to a specific page
  • Window.history.forward and Window.history.back() can let you navigate forward and backward by a page

Controlling Windows

  • Use window.open() to open a new window and window.close() to close it
  • There is also a moveTo and resizeTo method

Screen Information

  • The window.screen object has info on your browser screen 
  • More useful on mobile (detecting change in orientation, turn off device screen, etc)

The Document Object

  • Each window contains a document object
  • Cookies: small files saved locally to help save state
    • Made up a text file of name/value pairs
    • Create once using document.cookie  = ‘key=value’;
    • Cookies don’t overwrite, they append
    • Reassign by using same key with different value
    • Cookies are session by default but can persist by adding an expiry date
      • Add  "; expires=date" (date is a utc string) to the end of the cookie
    • Cookies can only be read by pages in the same directory and domain where the file was set

Timing Functions

  • Used for executing functions at intervals
    • window.setTimeout(), window.clearTimeout(), window.setInterval() 
  • Can be used for animations
  • requestAnimationFrame() is better
  • Using CSS for animations is best

Chapter 14: HTML5 APIs

The data- Attribute

  • Used to embed data using attributes that are ignored by the browser
    • Sole purpose is to be used by a JS program
    • Must start with data-, contain only lowercase letters, numbers, dots, hyphens, colon, or underscores and include an optional string value
    •  Superman
    • The data-powers attribute can be accessed using:
      • const superman = document.getElementById('hero'); const powers = superman.dataset.powers; << 'flight superSpeed'

HTML5 APIs

Help give access to hardware (cameras, GPS, batteries, etc) Here are a few examples:

  • HTML5 Web Storage (localStorage): similar to cookies (key/value storage) but with fewer restrictions
  • Geolocation API: used to find the geographical location of the device
    • It’s a property of the navigator object
  • Web Workers: allow processes to be run in the background so they don’t prevent the rest of the script from running. Especially useful for long running scripts
  • Websockets: allow two-way communication with a server (aka push messaging)
    • Uses wss:// instead of https://
  • Notifications: allows you to show messages using the operating system’s notifications
  • Multimedia:  allows JavaScript to control the playback of audio and video

Shims and Polyfills

  • Used to fix new API functionality on unsupported browsers. 
  • Shim: adds missing functionality to a browser
  • Polyfill: same as shim but also uses API commands as if it was supported natively
